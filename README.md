#Get that Dough

Tic-Tac-Toe game featuring real-time gameplay by players on seperate devices.

## Features

- AngularJS
- Fully responsive design & Mobile Optimized
- Firebase live database
- Interactive touch control
- Live updating
- CSS Animation


##Stack
```
AngularJS
Javascript
Firebase
HTML5 & CSS3

```
